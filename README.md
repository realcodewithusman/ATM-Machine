# ATM-Machine

A simple and responsive ATM Machine web app built using HTML, CSS, and JavaScript.

## Features
- Deposit money
- Withdraw money
- Real-time balance update
- Clean and responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Clone the repository
2. Open `index.html` in your browser
3. Start using the ATM simulation
